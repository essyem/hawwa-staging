Reference Number Format
Format Structure:
TRENDZ / [Department] / [Type] / [YearMonth][Sequence]
Components:
  • TRENDZ: Static prefix for all references
  • Department: 3-letter department code (e.g., HRD, FIN, OPR)
  • Type: 3-letter document type code (e.g., MOI, MOL, CRA)
  • Year: Last 2 digits of the year (e.g., 25 for 2025)
  • Month: 2-digit month (e.g., 08 for August)
  • Sequence: 4-digit sequential number starting from 1001
Examples:
  • TRENDZ/HRD/MOI/25081001 - First HR document for Ministry of Interior in August 2025
  • TRENDZ/FIN/TAX/25081002 - Second Finance document for Tax Authority in August 2025
  • TRENDZ/OPR/CON/25091001 - First Operations contract document in September 2025
